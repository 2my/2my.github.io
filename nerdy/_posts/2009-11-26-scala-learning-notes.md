implement  java interfaces</li><li>understands java  generics</li><li>uses   java  primitives<br /></li></ul>Leads you into   <span style="font-weight: bold;">functional  programming</span>.<br />Easy  to do easy  stuff, AND facilitates the hard stuff.<br /><span style="font-weight: bold;">More up2date</span> language - builds on java lessons, adds:<span style="font-weight: bold;"><br /></span><ul><li>Facilitates   DSL  with several short-hand notations</li><li>Traits (~multiple  inheritance)</li><li>++<br /></li></ul>One vision (Martin Odersky), not designed by   comitee<br />Pragmatic<br /><br /><br /><br />"I can honestly say if someone had shown  me  the Programming Scala  book by by Martin Odersky, Lex Spoon &amp;  Bill  Venners back in 2003 I’d  probably have never created Groovy." —  James  Strachan<br /><br />"If I were  to pick a language to use today other  than  java, it would be Scala" —  James Gosling (from before he left  Oracle)<br /><br
